# ❤️ Heart Disease Dashboard

An interactive **Heart Disease Analysis Dashboard** designed to explore patient data, survival rates, and key health indicators using data visualization techniques.

---

## 📊 Overview

This dashboard provides insights into:
- Survival rate (Alive %)
- Average age of patients
- Total alive vs total deaths
- Age group analysis
- Serum level trends
- Diabetes impact on survival

---

## 🚀 Features

- 📈 Interactive and visually rich dashboard
- 🧑‍⚕️ Patient survival analysis (Alive vs Death)
- 📊 Age group segmentation (40-50, 51-60, 61-70, 71+)
- 🧪 Health metrics visualization (Serum levels)
- 🩺 Diabetes impact comparison
- 👨‍👩‍👧 Gender-based filtering

---

## 🛠️ Tools & Technologies Used

- Power BI /  Excel / SQL
- Data Cleaning & Transformation
- Data Visualization

---

## 📂 Dataset

- Dataset is **public / open-source**
- Contains patient health records including:
  - Age
  - Gender
  - Serum levels
  - Diabetes status
  - Survival outcome

## 📌 Key Insights

- 68% patients survived (Alive Percentage)
- Highest survival observed in age group **51-60**
- Serum levels remain relatively stable across age groups
- Diabetes shows a noticeable impact on survival rates
- Majority of patients fall in the 51–70 age range
